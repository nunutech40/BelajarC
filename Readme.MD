# Dokumentasi Belajar Bahasa C

## create file
```bat
touch namafile+ext
```

## compile file c
```bat
clang hello.c -o hello
```

## cek file
```bat
ls
```

## get info file
```bat
file namafile
```

## hapus file
```bat
rm namafile
```

## Menggunakan CMake 
Karena sudah di configurasi CMakenya, cara buildnya tinggal:
```bat
make
```

then, contoh build file hello
```bat
./hello
```
